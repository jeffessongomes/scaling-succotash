## Tasks

### Task 1 — Adicionar índices ao schema.prisma
- **Arquivo:** `apps/api/prisma/schema.prisma`
- **Mudanças:**
  - `Question`: adicionar `@@index([quizId, order])`
  - `GameSession`: adicionar `@@index([status])`
  - `GameParticipant`: adicionar `@@index([sessionId])` após `@@unique`
  - `GameAnswer`: adicionar `@@index([sessionId])` e `@@index([questionId])` após `@@unique`
- **Verificação:** `pnpm --filter @azimute/api type-check`
- **Nota TDD:** Schema não tem teste unitário — validação via migration (Task 3) e type-check
