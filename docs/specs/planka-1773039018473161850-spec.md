## Edge Cases

1. **Token JWT expirado** â†’ 401, mensagem: `"Token expirado"`
2. **Token JWT malformado ou assinatura invĂˇlida** â†’ 401, mensagem: `"Token invĂˇlido"`
3. **Authorization header com formato errado** (ex: sem `Bearer `) â†’ 401
4. **PATCH com email jĂˇ usado por outro usuĂˇrio** â†’ 409 Conflict
5. **PATCH/DELETE de usuĂˇrio inexistente** â†’ 404 Not Found
6. **PATCH sem nenhum campo permitido no body** â†’ 400 Bad Request (Zod valida ao menos 1 campo)
7. **POST /users sem `password`** â†’ 400 (Zod requere)
8. **DELETE do Ăşnico ADMIN ativo** â†’ permitido (nĂŁo hĂˇ proteĂ§ĂŁo contra isso; responsabilidade do operador)

10. **PATCH com `role` no body** â†’ ignorado silenciosamente (campo nĂŁo permitido no schema Zod)
